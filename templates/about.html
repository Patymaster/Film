Про сайт

MovieBase — сайт-каталог фільмів, створений для ознайомлення з Flask. Реалізовано: маршрутизацію, шаблон Jinja2, сторінки каталогу, фільтрацію/сортування та форму зворотного зв’язку.

  • Flask — серверна логіка та маршрути
  • Jinja2 — HTML-шаблони та наслідування base.html
  • Static — CSS у папці static
  • Server — розгорнуто на Azure
